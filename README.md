# 医院预约App
## 实现功能
+ 用户界面登录、注册
+ 专家列表查询
+ 使用组件实现时间选择器和Dialog定制
+ 预约时间冲突避免
+ 支付宝支付
+ 医生登录，编辑个人信息
+ 取消预约

+ 界面实现的不够美观，可以展现的更高级感  
+ 支付宝功能闪退，无法完全支付，并且支付方式太单一
